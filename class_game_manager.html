◆ UpdatePlayerFloorText()

void GameManager.UpdatePlayerFloorText ( List< GameObject >  updatePlayerList,
List< int >  updateFloors 
)

This method is called by CheckPlayerFloors() when one or more players entered a new floor. It calls RpcUpdatePlayerFloorText(List<GameObject>, List<int>) that updates the player info UI for every client.

The two arguments are meant to be use like a dictionary, but Mirror RPCs cannot handle Dictionary<GameObject, int>.

Parameters
updatePlayerListA list of integers that represents the new floors that should be displayed in the player info UI.
updateFloorsA list of references to the player game objects that entered a new floor.
