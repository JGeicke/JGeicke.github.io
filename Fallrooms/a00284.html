◆ DebugIsPlayModeTest()

void SoundManager.DebugIsPlayModeTest ( )

This is called in PlayMode-Test to deactivate access to audio clips in the SoundManager.

Trying to access missing audio clips in the PlayMode tests throws an Exception.

This is an easier workaround when trying to test features that need the soundmanager but are not features of it.

It should only be used inside of playmode tests to not disrupt the flow of the game.
