Detailed Description

This class manages the UI elements in the current scene.

It is used to end the game & set the volume in the SoundManager

Member Function Documentation

◆ ActivatePlayerInfo()

void UIManager.ActivatePlayerInfo ( int  count)

This method activates the player info in the top right corner. For every player one ui element is activated.

This method is a RPC, so it gets called for every client.

It gets called in StartGame()

Mirrors ClientRPCs are not testable with the Unity Test Runner.

Parameters
idsA list of pseudo ids that are displayed in the player name text

◆ DebugInitTestUI()

void UIManager.DebugInitTestUI ( )

Initializes a test ui for the playmode tests.

If there are no UI elements, the UIManager throws a NullReferenceException.

The UIManager needs any UI-element to work properly so this method initializes the basic ui.

◆ QuitGame()

void UIManager.QuitGame ( )

Quits the game.

This method is called when the player presses the "exit game"-button.

It only works in the build version of the game.
